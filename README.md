# 📊 Customer Churn Analysis

An end-to-end **Data Analytics Project** that analyzes customer churn using **Python, MySQL, and Power BI**. The project focuses on identifying the key factors influencing customer churn and provides actionable business recommendations to improve customer retention.

---

## 📌 Project Overview

Customer churn is one of the biggest challenges faced by telecom companies. Understanding why customers leave helps businesses improve customer satisfaction and increase revenue.

In this project, I performed:

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- SQL Analysis using MySQL
- Interactive Dashboard Development in Power BI
- Business Insights and Recommendations

---

## 🎯 Business Objective

The objective of this project is to:

- Analyze customer behavior and churn patterns.
- Identify high-risk customer segments.
- Discover key factors contributing to customer churn.
- Provide business recommendations to improve customer retention.

---

## 📂 Dataset

**Dataset:** IBM Telco Customer Churn Dataset

The dataset contains information about:

- Customer Demographics
- Services Subscribed

# 🔍 Key Business Insights

- Approximately **26.5%** of customers have churned.
- Customers with **Month-to-Month contracts** have the highest churn rate.
- **Fiber Optic** users churn more frequently than DSL users.
- Customers using **Electronic Check** show higher churn.
- Customers with longer tenure are more likely to remain with the company.
- Long-term contracts contribute significantly to customer retention.

---

# 💡 Business Recommendations

- Encourage customers to switch from Month-to-Month to long-term contracts.
- Launch retention campaigns for customers with high churn risk.
- Improve customer support for Fiber Optic subscribers.
- Review and improve the Electronic Check payment experience.
- Offer loyalty rewards and personalized discounts to long-term customers.

# 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/KHUSHIJAISWAL11/customer-churn-analysis.git
```

### 2. Install Required Libraries

```bash
pip install -r requirements.txt
```

### 3. Run the Jupyter Notebooks

Open the notebooks in Jupyter Notebook or JupyterLab.

### 4. Execute SQL Scripts

Run the SQL files in MySQL Workbench in the following order:

1. `01_create_database.sql`
2. `02_create_table.sql`
3. `03_import_data.md` (follow the instructions)
4. `04_analysis_queries.sql`

### 5. Open Power BI Dashboard

Open the `.pbix` file available in the `reports` folder.
